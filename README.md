## English

This repository contains a simple Java console program that prints a centered pyramid (triangle) pattern based on a user-provided height. It's great for practicing nested loops, console formatting, and basic Java structure.

---

### 🧱 Example Output

```text
    *
   ***
  *****
 *******
*********
```

---

### ⭐ Features

* Prompts the user for pyramid height
* Uses nested loops to print spaces and stars
* Produces a centered pyramid
